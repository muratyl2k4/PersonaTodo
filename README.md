# Persona ToDo

Bu proje, **Software Persona** bünyesindeki staj programı kapsamında, modern Javascript ve React yetkinliklerini pekiştirmek amacıyla geliştirilmiş bir örnek ToDo uygulamasıdır.

## 📝 Proje Hakkında

Uygulama, temel bir görev yönetim sisteminin ötesine geçerek; modern arayüz tasarımı, dinamik durum yönetimi ve detaylı veri takibi gibi konseptleri bir araya getirmektedir. Staj süresince öğrenilen kavramların bütüncül bir şekilde kullanımını hedefler.

### Öne Çıkan Özellikler:
- **Görev Yönetimi**: Başlık, detaylı açıklama ve bitiş tarihi içeren görev yapısı.
- **Dinamik Arayüz**: Popup modal formu ve genişleyebilir görev listesi.
- **Zaman Takibi**: Oluşturulma ve tamamlanma zamanlarının otomatik kaydı.
- **Deadline Sistemi**: Bitiş tarihine kalan sürenin hesaplanması ve görsel uyarılar.
- **Hafıza**: `localStorage` entegrasyonu ile yerel veri saklama.

## 🛠️ Kullanılan Teknolojiler

- **React 19**: Bileşen tabanlı kullanıcı arayüzü geliştirme.
- **Vite**: Hızlı geliştirme ve derleme ortamı.
- **Tailwind CSS v4**: Modern mimari ile stil yönetimi.
- **Lucide React**: İkon kütüphanesi.
- **Netlify**: Sürekli dağıtım (CD) ve canlı yayın.

## ⚙️ Kurulum ve Çalıştırma

1. Projeyi klonlayın.
2. `npm install` komutu ile bağımlılıkları yükleyin.
3. `npm run dev` komutu ile yerel sunucuyu başlatın.
